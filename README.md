# Register Student
program designed for student registration.

# Sescription
The program is designed to facilitate student registration, student lists, quick searches, updating student data, and deleting student data.

The program has five functions plus error handling. Each function has a name that clearly indicates its function, making it easy and simple to use.

The first option is to add student data.
The second is to display the list of already registered students.
The third is to search for a student using their ID.
The fourth is to update the data of an already registered student.
The fifth deletes the data and the student from the system.
And finally, the last option is to exit.

# Getting Started
python 3.12 from now on.

Windows 10 from now on.

# installing
For the program to work correctly, you must install the two .py folders: main.py and functions.py. 
# Executing program

After that, to run the program, you must start running it from the main.py folder.
